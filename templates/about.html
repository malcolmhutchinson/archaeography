Welcome to archaeography.nz

Malcolm Hutchinson, April 2017.

My name is Malcolm Hutchinson. I'm an archaeologist, and a Geographic Information Scientist. I live and work in Hamilton, New Zealand.

This website is a test bed for a bunch of experiments in applying computational theory to archaeological data. I build relational databases and web interfaces, and I experiment with analytical methods, using archaeological records as my test subject. The main sections of the website introduce the research I do here.

The website is built using the Django web framework, and I write code in Python to collect and present archaeological records, and to perform statistical and geometric analysis on them. You can see an example of this analysis here.
